# 渠道

渠道代表与上游 AI 提供商的连接。

## 什么是渠道？

渠道是配置好的与上游 AI 提供商（OpenAI、Azure、Anthropic 等）的连接。管理员可以设置多个渠道以实现冗余和负载均衡。

## 渠道类型

| 类型 | 说明 |
|------|------|
| OpenAI | 直连 OpenAI API |
| Azure | Azure OpenAI 服务 |
| Anthropic | Claude API |
| Google | Gemini API |
| AWS Bedrock | Amazon Bedrock |
| 自定义 | 自定义兼容 OpenAI 的端点 |

## 渠道配置

每个渠道可以配置：
- **Base URL**：提供商端点
- **API Key**：认证密钥
- **模型**：支持的模型
- **分组**：用户分组分配
- **优先级**：负载均衡优先级

## 负载均衡

当多个渠道支持同一模型时，AronaApi 根据渠道优先级和可用性自动分配请求。
