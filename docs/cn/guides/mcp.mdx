# 文档 MCP

Mintlify 文档站可提供 MCP 入口，方便在 Cursor / Claude / VS Code 中检索本文档。

## 端点

```text
https://apidocs.lingshichat.top/mcp
```

## 客户端示例

### Cursor

```json
{
  "mcpServers": {
    "arona-docs": {
      "url": "https://apidocs.lingshichat.top/mcp"
    }
  }
}
```

### Claude Code

```bash
claude mcp add arona-docs https://apidocs.lingshichat.top/mcp
```

## 说明

- 文档 MCP 用于检索 **本帮助站内容**
- 它 **不是** 模型推理网关；调用 `grok-4.5` 仍走 `https://sub2api.lingshichat.top/v1`
