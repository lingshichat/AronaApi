# Model Context Protocol (MCP)

AronaApi supports the [Model Context Protocol](https://modelcontextprotocol.io), enabling AI assistants to discover and interact with your API through standardized tools.

## What is MCP?

The Model Context Protocol is an open standard that enables AI systems to interact with external tools, data sources, and services through a unified interface.

## MCP Server Endpoint

```
https://apidocs.lingshichat.top/mcp
```

## Supported Clients

<Tabs>
  <Tab title="Claude Code">
    ```bash
    claude mcp add arona-api https://apidocs.lingshichat.top/mcp
    ```
  </Tab>
  <Tab title="Cursor">
    Add to your Cursor MCP settings:
    ```json
    {
      "mcpServers": {
        "arona-api": {
          "url": "https://apidocs.lingshichat.top/mcp"
        }
      }
    }
    ```
  </Tab>
  <Tab title="VS Code">
    Install the MCP extension and add the server URL.
  </Tab>
</Tabs>

## Available Tools

The MCP server provides tools for:

- **Search Documentation**: Query the AronaApi documentation
- **Query Filesystem**: Browse available API endpoints and models

## Authentication

The MCP server supports both public and authenticated access:

- **Public**: `/mcp` - Basic documentation access
- **Authenticated**: `/authed/mcp` - Full API access with user credentials

For authenticated access, configure OAuth client credentials in the dashboard.
