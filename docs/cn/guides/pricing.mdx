# 定价

AronaApi 采用基于 Token 使用量的灵活定价系统。

## 定价模型

按每 1K Token 计费，不同模型价格不同：

| 模型 | 输入 (每 1K) | 输出 (每 1K) |
|-------|---------------|----------------|
| GPT-4o | $0.005 | $0.015 |
| GPT-4o-mini | $0.00015 | $0.0006 |
| Claude 3.5 Sonnet | $0.003 | $0.015 |
| Gemini 1.5 Pro | $0.0035 | $0.0105 |

## 查看定价

访问 [定价页面](https://api.lingshichat.top/pricing) 获取完整、最新的模型定价列表。

## 计费

- 使用量实时跟踪
- 可按用户或分组配置配额
- 支持多种货币显示
