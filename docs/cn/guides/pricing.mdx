# 定价

计费以控制台实时展示为准。不同分组、模型、订阅方案价格可能不同。

## 查看最新价格

1. 登录 [控制台](https://sub2api.lingshichat.top)
2. 打开定价 / 模型 / 订阅相关页面
3. 以页面显示的单价与配额规则为准

## 计费常见构成

- 输入 Token
- 输出 Token
- 缓存命中（如有）
- 图片 / 视频等特殊能力（如开通）

## 使用建议

- 编程代理场景优先固定模型 `grok-4.5`，方便估算成本
- 为不同项目拆分 API Key，便于对账
- 关注控制台用量与余额提醒
