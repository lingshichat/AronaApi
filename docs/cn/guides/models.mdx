## 模型路由

模型根据模型 ID 自动路由到正确的提供商。只需在 API 请求中使用模型名称：

```bash
curl https://api.lingshichat.top/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -d '{
    "model": "claude-3-5-sonnet-20240620",
    "messages": [{"role": "user", "content": "你好"}]
  }'
```

## 自定义模型

管理员可以在控制台中配置自定义模型映射，将模型请求重定向到特定渠道。
