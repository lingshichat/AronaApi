# 身份验证

AronaApi 使用 API Key 进行身份验证。所有 API 请求都必须包含有效的 API Key。

## API Key 格式

API Key 通过 `Authorization` 头部以 Bearer 方式传递：

```bash
Authorization: Bearer YOUR_API_KEY
```

## 获取 API Key

1. 登录 [控制台](https://api.lingshichat.top)
2. 在侧边栏中导航到 **API Key**
3. 点击 **创建 Key** 生成新 Key
4. 复制 Key 并妥善保存

<Warning>
  请妥善保管你的 API Key，切勿在客户端代码或公开仓库中暴露。
</Warning>

## 使用示例

```bash
curl https://api.lingshichat.top/v1/chat/completions \
  -H "Authorization: Bearer sk-xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "你好"}]
  }'
```

## 速率限制

API 请求根据账号等级受到速率限制：

| 等级 | 每分钟请求数 |
|------|-------------|
| 免费 | 60 |
| 标准 | 600 |
| 高级 | 6,000 |

所有响应都包含速率限制头部：

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1699999999
```
