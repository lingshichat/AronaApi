# Arona API

Arona API 是一个统一的 AI API 中转站，把上游模型能力聚合成 OpenAI / Claude 兼容接口，方便在 Grok Build、Codex、Claude Code 等客户端里一键接入。

## 站点信息

| 项目 | 地址 |
| --- | --- |
| 控制台 | [https://sub2api.lingshichat.top](https://sub2api.lingshichat.top) |
| API Base URL | `https://sub2api.lingshichat.top/v1` |
| 推荐模型 | `grok-4.5` |
| 文档 | [https://apidocs.lingshichat.top/cn](https://apidocs.lingshichat.top/cn) |
| 联系 | QQ 群 `1091297190` |

## 核心能力

- **OpenAI 兼容**：`/v1/chat/completions`、`/v1/responses`、`/v1/models`
- **Claude 兼容**：`/v1/messages`
- **Grok 平台能力**：对话、图片、视频（视分组/上游开通情况）
- **密钥与分组**：控制台创建 API Key，绑定可用模型分组

## 三分钟上手

<Steps>
  <Step title="注册并创建 API Key">
    打开 [控制台](https://sub2api.lingshichat.top) 注册登录，在密钥页面创建 API Key。
  </Step>
  <Step title="确认 Base URL 与模型">
    Base URL 使用 `https://sub2api.lingshichat.top/v1`，模型使用 `grok-4.5`。
  </Step>
  <Step title="配置你的客户端">
    按 [客户端配置总览](/cn/clients/overview) 选择 Grok Build / Codex / Claude Code。
  </Step>
</Steps>

## 最小请求示例

```bash
curl https://sub2api.lingshichat.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "grok-4.5",
    "messages": [
      {"role": "user", "content": "你好，用一句话介绍你自己"}
    ]
  }'
```
