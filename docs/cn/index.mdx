# AronaApi

AronaApi is a unified AI API gateway that aggregates 40+ upstream AI providers behind a single OpenAI-compatible API endpoint.

## What is AronaApi?

AronaApi provides a unified interface for accessing multiple AI models from different providers including OpenAI, Claude, Gemini, Azure, AWS Bedrock, and more.

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Get API Key">
    Sign up at [console](https://api.lingshichat.top) and generate an API key
  </Step>
  <Step title="Configure Base URL">
    Point your client to the AronaApi base URL
  </Step>
  <Step title="Start Using">
    Use any OpenAI-compatible client with your API key
  </Step>
</Steps>

## API Endpoint

```bash
https://api.lingshichat.top/v1/chat/completions
```

Replace your standard OpenAI base URL with AronaApi's URL and start using immediately.

## Model Context Protocol (MCP)

AronaApi supports the [Model Context Protocol](/guides/mcp), enabling AI assistants to interact with your API through standardized tools.
