# AronaApi

AronaApi 是一个统一的 AI API 网关，将 40+ 上游 AI 提供商聚合在单个兼容 OpenAI 的 API 端点之后。

## AronaApi 是什么？

AronaApi 提供了一个统一的接口，用于访问来自不同提供商的多种 AI 模型，包括 OpenAI、Claude、Gemini、Azure、AWS Bedrock 等。

## 快速开始

几分钟内即可开始使用：

<Steps>
  <Step title="获取 API Key">
    在 [控制台](https://api.lingshichat.top) 注册账号并生成 API Key
  </Step>
  <Step title="配置 Base URL">
    将客户端指向 AronaApi 的 Base URL
  </Step>
  <Step title="开始使用">
    使用任意兼容 OpenAI 的客户端配合你的 API Key
  </Step>
</Steps>

## API 端点

```bash
https://api.lingshichat.top/v1/chat/completions
```

将标准 OpenAI Base URL 替换为 AronaApi 的 URL，即可立即使用。

## Model Context Protocol (MCP)

AronaApi 支持 [Model Context Protocol](/cn/guides/mcp)，让 AI 助手可以通过标准化工具与你的 API 交互。
