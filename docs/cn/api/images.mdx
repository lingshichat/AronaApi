# 图片

使用 AI 模型生成和编辑图片。

## 生成图片

```bash
POST /v1/images/generations
```

### 请求

```json
{
  "model": "dall-e-3",
  "prompt": "日落时分的宁静山景",
  "size": "1024x1024",
  "quality": "standard",
  "n": 1
}
```

### 响应

```json
{
  "created": 1677721600,
  "data": [
    {
      "url": "https://.../generated-image.png",
      "revised_prompt": "日落时分的宁静山景，金色光芒"
    }
  ]
}
```

## 可用尺寸

| 尺寸 | 说明 |
|------|------|
| `1024x1024` | 正方形 |
| `1792x1024` | 横屏 |
| `1024x1792` | 竖屏 |

## 支持的模型

- `dall-e-3`
- `dall-e-2`
- 各种 Midjourney 模型（通过 Midjourney Proxy）
