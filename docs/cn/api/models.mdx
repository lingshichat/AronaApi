# 模型列表

```bash
GET /v1/models
```

## 请求

```bash
curl https://sub2api.lingshichat.top/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 说明

- 返回结果与 **当前 API Key / 分组** 有关
- 本文档主推模型：`grok-4.5`
- 客户端配置前建议先调用此接口确认模型 ID
