# 模型

AronaApi 通过统一接口提供来自多个提供商的 40+ AI 模型。

## 列出模型

```bash
GET /v1/models
```

### 响应

```json
{
  "object": "list",
  "data": [
    {
      "id": "gpt-4o",
      "object": "model",
      "created": 1677610602,
      "owned_by": "openai"
    },
    {
      "id": "claude-3-5-sonnet-20240620",
      "object": "model",
      "created": 1712003200,
      "owned_by": "anthropic"
    },
    {
      "id": "gemini-1.5-pro",
      "object": "model",
      "created": 1710532800,
      "owned_by": "google"
    }
  ]
}
```

## 支持的提供商

| 提供商 | 模型 |
|----------|--------|
| OpenAI | GPT-4o, GPT-4o-mini, GPT-4, GPT-3.5-Turbo |
| Anthropic | Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku |
| Google | Gemini 1.5 Pro, Gemini 1.5 Flash |
| Azure | GPT-4, GPT-3.5-Turbo |
| AWS Bedrock | Claude, Llama, Titan |
| DeepSeek | DeepSeek-V3, DeepSeek-R1 |
| 更多... | 在控制台查看完整列表 |

## 模型路由

AronaApi 根据模型 ID 自动将请求路由到对应的提供商，无需额外配置。

<Tip>
  在 [控制台](https://api.lingshichat.top) 查看完整的可用模型列表和定价。
</Tip>
