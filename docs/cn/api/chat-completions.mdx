# 对话补全

使用各种 AI 模型创建对话补全。

## 端点

```bash
POST /v1/chat/completions
```

## 请求体

```json
{
  "model": "gpt-4o",
  "messages": [
    {"role": "system", "content": "你是一个有用的助手。"},
    {"role": "user", "content": "你好！"}
  ],
  "temperature": 0.7,
  "max_tokens": 150,
  "stream": false
}
```

### 参数

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `model` | string | 是 | 要使用的模型 ID |
| `messages` | array | 是 | 消息列表 |
| `temperature` | number | 否 | 采样温度 (0-2) |
| `max_tokens` | integer | 否 | 最大生成 Token 数 |
| `stream` | boolean | 否 | 启用流式输出 |
| `top_p` | number | 否 | 核采样参数 |
| `frequency_penalty` | number | 否 | 频率惩罚 (-2 到 2) |
| `presence_penalty` | number | 否 | 存在惩罚 (-2 到 2) |

## 响应

```json
{
  "id": "chatcmpl-123",
  "object": "chat.completion",
  "created": 1677652288,
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "你好！有什么可以帮你的吗？"
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 19,
    "completion_tokens": 9,
    "total_tokens": 28
  }
}
```

## 流式响应

当 `stream: true` 时，响应使用 Server-Sent Events：

```
data: {"id":"chatcmpl-123","object":"chat.completion.chunk","created":1677652288,"model":"gpt-4o","choices":[{"index":0,"delta":{"role":"assistant"},"finish_reason":null}]}

data: {"id":"chatcmpl-123","object":"chat.completion.chunk","created":1677652288,"model":"gpt-4o","choices":[{"index":0,"delta":{"content":"你"},"finish_reason":null}]}

data: {"id":"chatcmpl-123","object":"chat.completion.chunk","created":1677652288,"model":"gpt-4o","choices":[{"index":0,"delta":{"content":"好"},"finish_reason":null}]}

data: [DONE]
```
