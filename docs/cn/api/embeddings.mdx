# 嵌入向量

```bash
POST /v1/embeddings
```

<Note>
  是否可用取决于你的 Key 所属平台/分组。Grok 分组未必开放 embeddings。
</Note>

## 请求示例

```bash
curl https://sub2api.lingshichat.top/v1/embeddings \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-3-small",
    "input": "hello world"
  }'
```

若返回不支持 / 404，说明当前分组未开通该能力。
