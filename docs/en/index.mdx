# Arona API

Arona API is a unified AI API gateway. It aggregates upstream models behind OpenAI- and Claude-compatible endpoints so you can plug in Grok Build, Codex, Claude Code, OpenCode, Pi Agent, and more with one base URL.

## Site info

| Item | Value |
| --- | --- |
| Console | [https://sub2api.lingshichat.top](https://sub2api.lingshichat.top) |
| API Base URL | `https://sub2api.lingshichat.top/v1` |
| Recommended model | `grok-4.5` |
| Docs | [https://apidocs.lingshichat.top/en](https://apidocs.lingshichat.top/en) |
| Contact | QQ group `1091297190` |

## Capabilities

- **OpenAI-compatible**: `/v1/chat/completions`, `/v1/responses`, `/v1/models`
- **Claude-compatible**: `/v1/messages`
- **Grok platform features**: chat, images, video (depends on group / upstream)
- **Keys and groups**: create API keys in the console and bind model groups

## Get started in three minutes

<Steps>
  <Step title="Register and create an API key">
    Open the [console](https://sub2api.lingshichat.top), sign in, and create an API key.
  </Step>
  <Step title="Confirm Base URL and model">
    Use `https://sub2api.lingshichat.top/v1` and model `grok-4.5`.
  </Step>
  <Step title="Configure your client">
    Follow [Client overview](/en/clients/overview) for Grok Build / Codex / Claude Code / OpenCode / Pi.
  </Step>
</Steps>

## Minimal request

```bash
curl https://sub2api.lingshichat.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "grok-4.5",
    "messages": [
      {"role": "user", "content": "Hello, introduce yourself in one sentence"}
    ]
  }'
```
