# Quickstart

This guide covers: register → create a key → verify the API → configure a client.

## Prerequisites

- An [Arona API console](https://sub2api.lingshichat.top) account
- A working API key (create on the keys page)
- A client: Grok Build / Codex / Claude Code / OpenCode / Pi Agent

## 1. Get an API key

1. Sign in at [https://sub2api.lingshichat.top](https://sub2api.lingshichat.top)
2. Open **API Keys**
3. Create a key and store it safely

<Warning>
  Treat API keys as secrets. Do not commit them to public repos or share them with untrusted parties.
</Warning>

## 2. Connection info

| Setting | Value |
| --- | --- |
| Console | `https://sub2api.lingshichat.top` |
| API Base URL | `https://sub2api.lingshichat.top/v1` |
| Recommended model | `grok-4.5` |
| Auth header | `Authorization: Bearer <YOUR_API_KEY>` |

## 3. Verify with cURL

### List models

```bash
curl https://sub2api.lingshichat.top/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Chat Completions

```bash
curl https://sub2api.lingshichat.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "grok-4.5",
    "messages": [
      {"role": "user", "content": "ping"}
    ]
  }'
```

### Responses API (recommended for Grok Build)

```bash
curl https://sub2api.lingshichat.top/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "grok-4.5",
    "input": "ping"
  }'
```

If you get normal JSON or a stream, the key and group are usable.

## 5. OpenAI SDK examples

### Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://sub2api.lingshichat.top/v1",
    api_key="YOUR_API_KEY",
)

resp = client.chat.completions.create(
    model="grok-4.5",
    messages=[{"role": "user", "content": "Hello"}],
)
print(resp.choices[0].message.content)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://sub2api.lingshichat.top/v1',
  apiKey: 'YOUR_API_KEY',
});

const resp = await client.chat.completions.create({
  model: 'grok-4.5',
  messages: [{ role: 'user', content: 'Hello' }],
});

console.log(resp.choices[0].message.content);
```

## Troubleshooting

| Symptom | Likely cause | Fix |
| --- | --- | --- |
| `401` | Bad key or missing Bearer | Check `Authorization` |
| `403` / group error | Key not bound to a usable group | Check key group in console |
| `model_not_found` | Wrong model or group lacks it | Use `grok-4.5` or `GET /v1/models` |
| Timeout / connect fail | Network or wrong host | Use `sub2api.lingshichat.top` |
