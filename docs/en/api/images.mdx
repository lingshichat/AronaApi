# Images

```bash
POST /v1/images/generations
```

<Note>
  Image capability depends on upstream and group configuration. Unsupported groups return an error.
</Note>

## Example

```bash
curl https://sub2api.lingshichat.top/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_IMAGE_MODEL",
    "prompt": "a calm mountain lake at sunrise",
    "size": "1024x1024"
  }'
```

Use model IDs from `GET /v1/models` and the console.
