# List models

```bash
GET /v1/models
```

## Request

```bash
curl https://sub2api.lingshichat.top/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Notes

- Result depends on the **current API key / group**
- Docs recommend: `grok-4.5`
- Call this before configuring clients if you are unsure of model IDs
