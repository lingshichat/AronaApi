# Models

AronaApi provides access to 40+ AI models from various providers through a unified interface.

## List Models

```bash
GET /v1/models
```

### Response

```json
{
  "object": "list",
  "data": [
    {
      "id": "gpt-4o",
      "object": "model",
      "created": 1677610602,
      "owned_by": "openai"
    },
    {
      "id": "claude-3-5-sonnet-20240620",
      "object": "model",
      "created": 1712003200,
      "owned_by": "anthropic"
    },
    {
      "id": "gemini-1.5-pro",
      "object": "model",
      "created": 1710532800,
      "owned_by": "google"
    }
  ]
}
```

## Supported Providers

| Provider | Models |
|----------|--------|
| OpenAI | GPT-4o, GPT-4o-mini, GPT-4, GPT-3.5-Turbo |
| Anthropic | Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku |
| Google | Gemini 1.5 Pro, Gemini 1.5 Flash |
| Azure | GPT-4, GPT-3.5-Turbo |
| AWS Bedrock | Claude, Llama, Titan |
| DeepSeek | DeepSeek-V3, DeepSeek-R1 |
| and more... | See console for full list |

## Model Routing

AronaApi automatically routes requests to the appropriate provider based on the model ID. No additional configuration is needed.

<Tip>
  Use the [console](https://api.lingshichat.top) to see the full list of available models and their pricing.
</Tip>
