# Embeddings

```bash
POST /v1/embeddings
```

<Note>
  Availability depends on your key’s platform/group. Grok groups may not expose embeddings.
</Note>

## Example

```bash
curl https://sub2api.lingshichat.top/v1/embeddings \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-3-small",
    "input": "hello world"
  }'
```

If you get unsupported / 404, the current group does not offer this capability.
