# Docs MCP

The Mintlify site can expose an MCP endpoint so Cursor / Claude / VS Code can search these docs.

## Endpoint

```text
https://apidocs.lingshichat.top/mcp
```

## Client examples

### Cursor

```json
{
  "mcpServers": {
    "arona-docs": {
      "url": "https://apidocs.lingshichat.top/mcp"
    }
  }
}
```

### Claude Code

```bash
claude mcp add arona-docs https://apidocs.lingshichat.top/mcp
```

## Notes

- Docs MCP searches **this help site**
- It is **not** the model inference gateway; call `grok-4.5` via `https://sub2api.lingshichat.top/v1`
