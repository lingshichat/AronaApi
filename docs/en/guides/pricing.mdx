# Pricing

Billing always follows the live console. Prices can differ by group, model, and plan.

## See current prices

1. Sign in to the [console](https://sub2api.lingshichat.top)
2. Open pricing / models / subscription pages
3. Use the displayed unit prices and quota rules

## Common billing components

- Input tokens
- Output tokens
- Cache hits (if any)
- Image / video features (if enabled)

## Tips

- Pin model `grok-4.5` for coding agents to make cost easier to estimate
- Split keys per project for accounting
- Watch console usage and balance alerts
