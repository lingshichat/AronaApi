# Pricing

AronaApi uses a flexible pricing system based on token usage.

## Pricing Model

Pricing is calculated per 1K tokens and varies by model:

| Model | Input (per 1K) | Output (per 1K) |
|-------|---------------|----------------|
| GPT-4o | $0.005 | $0.015 |
| GPT-4o-mini | $0.00015 | $0.0006 |
| Claude 3.5 Sonnet | $0.003 | $0.015 |
| Gemini 1.5 Pro | $0.0035 | $0.0105 |

## View Pricing

Visit the [Pricing Page](https://api.lingshichat.top/pricing) for the complete, up-to-date model pricing list.

## Billing

- Usage is tracked in real-time
- Quotas can be configured per user or group
- Supports multiple currency displays
