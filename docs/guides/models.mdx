## Model Routing

Models are automatically routed to the correct provider based on the model ID. Simply use the model name in your API request:

```bash
curl https://api.lingshichat.top/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -d '{
    "model": "claude-3-5-sonnet-20240620",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

## Custom Models

Administrators can configure custom model mappings in the console to redirect model requests to specific channels.
